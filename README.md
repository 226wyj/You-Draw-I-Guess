# You-Draw-I-Guess

An image classification system based on CIFAR-10.



项目依赖：

* pytorch
* PIL
* tqdm
* argparse
* streamlit



项目目录：

* data文件夹用于存放CIFAR-10数据





训练：

```python
python ./main.py --do_train 
```



测试：

```python
python ./main.py --do_eval
```



预测：

```python
python ./main.py --do_predict
```
