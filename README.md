# CIFAR_10-with-pytorch

一个Pytorch练习，实现CIFAR-10数据集的图像分类



项目依赖：

* pytorch
* PIL
* tqdm
* argparse
* streamlit



项目目录：

* data文件夹用于存放CIFAR-10数据





训练：

```python
python ./main.py --do_train 
```



测试：

```python
python ./main.py --do_eval
```



预测：

```python
python ./main.py --do_predict
```
